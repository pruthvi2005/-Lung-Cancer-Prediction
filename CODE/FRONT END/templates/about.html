About Project

Lung Cancer Prediction Using Machine Learning

This project leverages machine learning algorithms such as Random Forest, XGBoost, SVM, adaboost and a Voting Classifier to predict lung cancer risk levels. The model categorizes individuals into three risk categories: low, medium, and high, based on various health parameters and diagnostic data. By combining the strengths of multiple classifiers, the Voting Classifier improves prediction accuracy and robustness, making it a reliable tool for early detection and risk assessment in lung cancer diagnosis. The use of these advanced machine learning techniques ensures optimal performance and precise classification for better decision-making in healthcare.
